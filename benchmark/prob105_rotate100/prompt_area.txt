I would like you to implement a module named TopModule with the following
interface. All input and output ports are one bit unless otherwise
specified.

 - input  clk
 - input  load
 - input  ena  (2 bits)
 - input  data (100 bits)
 - output q    (100 bits)

The module should implement a 100-bit left/right rotator with synchronous load.

Primary objective: minimize silicon area and total hardware resource usage.

Design constraints:
- Use the most compact implementation possible.
- Avoid redundant multiplexers or unnecessary control logic.
- Use minimal combinational logic for rotation.
- Do not introduce extra registers beyond what is required.
- Sequential logic must be triggered on the positive edge of clk.
- Return only synthesizable SystemVerilog code for TopModule.
