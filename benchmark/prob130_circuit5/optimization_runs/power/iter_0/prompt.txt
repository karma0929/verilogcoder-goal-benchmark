I would like you to implement a module named TopModule with the following
interface. All input and output ports are one bit unless otherwise
specified.

 - input  a (4 bits)
 - input  b (4 bits)
 - input  c (4 bits)
 - input  d (4 bits)
 - input  e (4 bits)
 - output q (4 bits)

The module should implement a combinational circuit. Read the simulation
waveforms to determine what the circuit does, then implement it.

Primary objective: minimize dynamic power (switching activity and glitches).

Constraints:
- Prefer coding style that maps cleanly to multiplexers with minimal hazard/glitch risk.
- Avoid deep reconvergent logic and unnecessary toggling of internal nodes.
- Keep logic depth small and structure straightforward.
- No sequential logic.
- Output only the synthesizable SystemVerilog code for TopModule.
