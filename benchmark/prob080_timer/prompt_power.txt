I would like you to implement a module named TopModule with the following
interface. All input and output ports are one bit unless otherwise
specified.

 - input  clk,
 - input  load,
 - input  data (10 bits)
 - output tc

The module should implement a timer that counts down and asserts tc when
the count reaches 0.

Primary objective: minimize dynamic power and switching activity.

Design constraints:
- Avoid unnecessary toggling of internal signals.
- Prevent redundant comparisons or logic reconvergence.
- Ensure the counter stops toggling once it reaches 0.
- Use clean combinational logic for tc generation.
- Sequential logic must be triggered on the positive edge of clk.
- Return only synthesizable SystemVerilog code for TopModule.
