I would like you to implement a module named TopModule with the following
interface. All input and output ports are one bit unless otherwise
specified.

 - input  in     (8 bits)
 - output parity

The module should compute an even parity bit for the 8-bit input.
The parity bit must be the XOR of all 8 data bits.

Primary objective: minimize silicon area and gate count.

Guidelines:
- Use the most compact combinational implementation possible.
- Avoid redundant logic or duplicated XOR trees.
- Do not introduce registers or sequential logic.
- Return only synthesizable SystemVerilog code for TopModule.
