I would like you to implement a module named TopModule with the following
interface. All input and output ports are one bit unless otherwise
specified.

 - input  clk,
 - input  load,
 - input  data (10 bits)
 - output tc

The module should implement a timer that counts down and asserts tc when
the count reaches 0.

Primary objective: minimize critical path delay and maximize clock frequency.

Design constraints:
- Structure decrement and zero-detection logic to reduce logic depth.
- Avoid long combinational chains.
- Keep zero comparison efficient.
- The counter must stop at 0 until load is asserted again.
- Sequential logic must be triggered on the positive edge of clk.
- Return only synthesizable SystemVerilog code for TopModule.
